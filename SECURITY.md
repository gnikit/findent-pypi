# Security Policy

This repository is responsible for generating the Python wheels for `findent`
and distributing them on PyPI. It does not contain any of the source code of
`findent` itself. Consequently, any security vulnerabilities in the `findent`
tool should be reported to the original maintainers of `findent`, at:

- https://sourceforge.net/p/findent
- https://github.com/wvermin/findent
- or via the author's email, which can be found in the `findent` source code.

If you discover a security vulnerability in this repository itself, please report it
using the instructions below.

## Reporting a Vulnerability

If a vulnerability is detected please open an issue.
